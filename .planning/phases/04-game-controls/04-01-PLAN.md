    **Handler implementations (add to switch(action.type) in onMessage):**

    **FLIP_CARD** (per D-01, D-02, D-03):
    - Find pile by `action.pileId`; if not found, send ERROR code "PILE_NOT_FOUND"
    - Find card by `action.cardId` in pile.cards; if not found, send ERROR code "CARD_NOT_FOUND"
    - Call `takeSnapshot(this.gameState, sender.id)`
    - Toggle `card.faceUp = !card.faceUp`

    **PASS_CARD** (per D-04, D-05, D-06):
    - Get `senderHand = this.gameState.hands[sender.id]`; if not found, break
    - Find card index in senderHand by `action.cardId`; if not found, send ERROR code "CARD_NOT_IN_HAND"
    - Check `this.gameState.hands[action.targetPlayerId]` exists; if not, send ERROR code "PLAYER_NOT_FOUND"
    - Call `takeSnapshot(this.gameState, sender.id)`
    - Splice card from sender hand, set `card.faceUp = true`, push to target player hand

    **DEAL_CARDS** (per D-07, D-08, D-09):
    - Find draw pile (`id === "draw"`); get connected players (`players.filter(p => p.connected)`)
    - Calculate `needed = action.cardsPerPlayer * connectedPlayers.length`
    - If draw pile not found or `drawPile.cards.length < needed`, send ERROR code "INSUFFICIENT_CARDS", break
    - Call `takeSnapshot(this.gameState, sender.id)`
    - Round-robin: outer loop `i < action.cardsPerPlayer`, inner loop over `connectedPlayers` — pop card from draw pile, set `card.faceUp = true`, push to `hands[player.id]`
    - Set `this.gameState.phase = "playing"`

    **SHUFFLE_PILE** (per D-12, reuses existing `shuffle` function):
    - Find pile by `action.pileId`; if not found, send ERROR code "PILE_NOT_FOUND"
    - Call `takeSnapshot(this.gameState, sender.id)`
    - Set `pile.cards = shuffle(pile.cards)`

    **RESET_TABLE** (per D-15, D-16):
    - Find draw pile
    - Collect all cards from all hands: `for (const hand of Object.values(this.gameState.hands)) { drawPile.cards.push(...hand.splice(0)); }`
    - Collect all cards from non-draw piles: `for (const pile of this.gameState.piles) { if (pile.id !== "draw") { drawPile.cards.push(...pile.cards.splice(0)); } }`
    - Set `drawPile.faceUp = false`
    - Set all cards faceUp to false: `drawPile.cards.forEach(c => { c.faceUp = false; })`
    - Reshuffle: `drawPile.cards = shuffle(drawPile.cards)`
    - Set `this.gameState.phase = "setup"`
    - Clear all snapshots: `this.gameState.undoSnapshots = {}`
    - Note: No takeSnapshot before reset — reset is irreversible per D-15 ("This can't be undone")

    **UNDO_MOVE** (per D-17, D-18, D-19):
    - Get snapshot: `const snap = this.gameState.undoSnapshots[sender.id]`
    - If no snapshot, break (no-op, no error)
    - Replace state: `this.gameState = snap` — note that `snap.undoSnapshots` was stripped to `{}` by takeSnapshot, so after restore the sender's canUndo is immediately false
    - Note: Do NOT call takeSnapshot before undo — undo-of-undo is not supported per D-18

This ensures undo works for any card action per D-17. Do NOT snapshot before SHUFFLE_DECK (it's a setup action, not a card move).
