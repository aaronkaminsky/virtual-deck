    1. Extract `{ cardId, fromZone, fromId, toZone, toId }` from action
    2. If `fromZone === "hand"` and `fromId !== sender.id`, send ERROR with code "UNAUTHORIZED_MOVE", message "Cannot move another player's cards", and break
    3. If `toZone === "hand"` and `toId !== sender.id`, send ERROR with code "UNAUTHORIZED_MOVE", message "Cannot place cards in another player's hand", and break
    4. Locate source: if `fromZone === "hand"`, source = `this.gameState.hands[fromId]`; if `fromZone === "pile"`, source = `this.gameState.piles.find(p => p.id === fromId)?.cards`
    5. If source undefined, send ERROR code "HAND_NOT_FOUND" or "PILE_NOT_FOUND" and break
    6. Find card index in source array by cardId. If -1, send ERROR code "CARD_NOT_IN_SOURCE" and break
    7. `const card = source.splice(idx, 1)[0]`
    8. Locate destination: if `toZone === "hand"`, dest = `this.gameState.hands[toId]` (create empty array if missing); if `toZone === "pile"`, dest = `this.gameState.piles.find(p => p.id === toId)?.cards`
    9. If dest undefined, send ERROR code "PILE_NOT_FOUND" and break
    10. `dest.push(card)`
