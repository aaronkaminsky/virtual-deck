### Implementation Deviation (Not a Gap)

The PLAN specified using `autoFocus` prop on the Top Button element to achieve focus-on-open. The actual implementation uses `initialFocus={topButtonRef}` on `Dialog.Popup` paired with `ref={topButtonRef}` on the Top Button. This requires `Button` to be wrapped in `React.forwardRef` (which it is — see `button.tsx` line 44–56). The observable truth (Enter confirms Top) is met by either approach. The SUMMARY documents this as an intentional deviation made during execution.

`button.tsx` was also modified (added `React.forwardRef` wrapper and `Button.displayName`) — this was not in the PLAN's `files_modified` list, but is listed in the SUMMARY. The change is a backwards-compatible enhancement that does not affect other callers.
