## Interaction Contract

### Sort Button (HandZone)

- **Placement:** After Eye/EyeOff toggle in hand header row (`flex items-center gap-2 px-4 mb-1`)
- **Element:** `<Button variant="ghost" className="h-7 w-7 p-0">` containing `<ArrowUpDown className="w-4 h-4" />`
- **Cycle:** Original → By Suit → By Rank → Original (click to advance)
- **Active state:** `className={cn('w-4 h-4', sortMode !== 'original' ? 'text-primary' : 'text-muted-foreground')}` on icon
- **Tooltip:** Dynamic text showing current mode and hint to next:
  - Original: `"Sort: Original order — click for By Suit"`
  - By Suit: `"Sort: By suit (♠ ♣ ♦ ♥) — click for By Rank"`
  - By Rank: `"Sort: By rank (2→A) — click for Original order"`
- **Disabled state:** Never disabled (hand may be empty, but button is always rendered per the existing Eye/EyeOff precedent)
- **No text label:** Icon-only, tooltip provides semantic context
- **Source:** CONTEXT.md D-04, D-05, D-06

### Select All Button (PileZone)

- **Placement:** Inside existing `<div className="flex gap-1 mt-1">` controls row (alongside Eye/EyeOff and Shuffle buttons)
- **Element:** `<Button variant="ghost" className="h-7 w-7 p-0">` containing `<SquareCheck className="w-4 h-4" />`
- **Active state:** None — button has no persistent toggle; icon always `text-muted-foreground`
- **Disabled state:** `disabled` attribute when pile has no cards (guard: `topCard` is null or not a `Card` type)
- **Tooltip:** `"Select all cards in pile"` (static)
- **Behavior on click:** Calls `onSelectAll([topCard.id], 'pile', pile.id)` — top card only (see RESEARCH.md Pattern 4 and Pitfall 4)
- **Source:** CONTEXT.md D-07, D-08, D-09; RESEARCH.md Pattern 4

### Select All Button (SpreadZone)

- **Placement:** In the same button row as the existing eye icon button (below the zone body)
- **Element:** `<Button variant="ghost" className="h-7 w-7 p-0">` containing `<SquareCheck className="w-4 h-4" />`
- **Active state:** None — always `text-muted-foreground`
- **Disabled state:** `disabled` attribute when spread zone has zero cards
- **Tooltip:** `"Select all cards in zone"` (static)
- **Behavior on click:** Calls `onSelectAll(faceUpCards.map(c => c.id), 'pile', pile.id)` — all cards in zone
- **Source:** CONTEXT.md D-07, D-08, D-10; RESEARCH.md Pitfall 5
