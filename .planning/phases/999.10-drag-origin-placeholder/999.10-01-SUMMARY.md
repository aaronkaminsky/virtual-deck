# Phase 999.10 Plan 01: Drag Origin Placeholder Summary

**One-liner:** Dashed card-shaped outline replaces invisible opacity-0 card at drag origin in both DraggableCard (pile) and SortableHandCard (hand).

## What Was Built

Replaced `opacity: isDragging ? 0 : 1` with a conditional JSX content branch in two components:

- **DraggableCard** (`src/components/DraggableCard.tsx`): When `isDragging`, renders `<div className="w-[80px] h-[112px] rounded-lg border border-dashed border-border" />` — an 80x112 dashed outline matching the PileZone empty-pile visual exactly.
- **SortableHandCard** in **HandZone** (`src/components/HandZone.tsx`): When `isDragging`, renders `<div className="w-[63px] h-[88px] rounded-md border border-dashed border-border" />` — a 63x88 dashed outline matching CardFace/CardBack inner dimensions so the hand strip layout does not shift.

Both changes: removed `opacity` from the style object, kept `transform`/`touchAction` (DraggableCard) and `transform`/`transition`/`touchAction` (SortableHandCard), left `ref={setNodeRef}`/`{...listeners}`/`{...attributes}` on the outer wrapper unchanged.
